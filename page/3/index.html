重点内容1：复合类型

变量的声明语句，是由基类型（int,float,char等）和其后面跟的一系列声明符（declarator）构成。每一个声明符将它后面的变量和基类型绑定起来，它只对紧跟着它的变量负责；基类型对它后面的所有变量负责。

重点内容2：初始化时推荐使用{}初始化，但是要注意列表初始化（即使用{}初始化）不允许窄化操作。

摘自 Bjarne Stroustrup’s “The C++ Programming Language 4th Edition” (section 6.3.5):

List initialization does not allow narrowing (§iso.8.5.4). That is:

  • An integer cannot be converted to another integer that cannot hold its value. For example, char to int is allowed, but not int to char.
  • A floating-point value cannot be converted to another floating-point type that cannot hold its value. For example, float to double is allowed, but not double to float.
  • A floating-point value cannot be converted to an integer type.
  • An integer value cannot be converted to a floating-point type.
