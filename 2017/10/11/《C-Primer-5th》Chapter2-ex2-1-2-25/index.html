重点内容1：初始化时推荐使用{}初始化，但是要注意列表初始化（即使用{}初始化）不允许窄化操作。

摘自 Bjarne Stroustrup’s “The C++ Programming Language 4th Edition” (section 6.3.5):

List initialization does not allow narrowing (§iso.8.5.4). That is:

  • An integer cannot be converted to another integer that cannot hold its value. For example, char to int is allowed, but not int to char.
  • A floating-point value cannot be converted to another floating-point type that cannot hold its value. For example, float to double is allowed, but not double to float.
  • A floating-point value cannot be converted to an integer type.
  • An integer value cannot be converted to a floating-point type.

Example:

The only situation where = is preferred over {} is when using auto keyword to get the type determined by the initializer.

Example:

结论：初始化优先考虑花括号{}，除非有其他原因。

重点内容2：当定义变量而没有对变量初始化，该变量会被默认初始化。默认初始化的值取决于该变量的类型和其所在位置（全局变量还是函数里的局部变量）。
  • 定义在函数外的全局变量如果没有显式初始化，会被默认初始化为0。
  • 定义在函数内的内置类型变量如果没有显式初始化，不会被默认初始化。此时该变量很危险，因为不知道它是什么值。
  • 定义在函数内的类（class）类型变量如果没有显式初始化，它是否能被默认初始化，取决于该类。

Exercise 2.5

Determine the type of each of the following literals. Explain the differences among the literals in each of the four examples:

(a) ‘a’, L’a’, “a”, L”a”

(b) 10, 10u, 10L, 10uL, 012, 0xC

(c) 3.14, 3.14f, 3.14L

(d) 10, 10u, 10., 10e-2

  • (a) ‘a’表示字符, L’a’表示宽字符, “a”，双引号表示字符串, L”a”，表示宽字符串
  • (b) 10 表示十进制整型, 10u表示10进制无符号整型，10L表示10进制长整型, 10uL表示10进制无符号长整型, 012表示8进制整数, 0xC表示16进制整数
  • (c) 3.14表示浮点数, 3.14f表示单精度浮点数, 3.14L表示long double浮点数
  • (d) 10表示整数, 10u表示无符号整数, 10.表示浮点数, 10e-2表示科学计数法

Exercise 2.10

  • global_str是string类型且是全局变量，会默认初始化为空字符串。
  • global_int是int类型变量且是全局变量，会默认初始化为0。
  • local_int虽是定义在函数内的局部变量且是内置类型，所以不会默认初始化。任何对其的操作，比如拷贝，打印都是危险的。
  • local_str虽是定义在函数内的局部变量，但由于string类自身定义了默认初始化，所以会被默认初始化为空字符串。
