Posted on 2017-11-19 | In C++ Primer 5th Edition Exercises
重点内容1：指针数组（数组的元素是指针）和数组指针（指向数组的指针）

技巧，从右向左读：

  • 先定义了一个包含10个元素的数组ptrs，它的元素类型是int *。
  • 先定义了一个包含10个元素的数组refs,它的元素类型是int &,但是数组的元素只能是对象，不能是引用，所以此条语句错误。

技巧，从内向外读：

  • 首先读小括号里面的，可知，Parray是一个指针；再读小括号右边，可知Parray指向的是长度为10的数组；再读小括号左边，可知，该数组元素类型是int。所以，Parray是指向包含10个int类型的数组的指针。
  • 同理，可知arrRef是指向包含10个int类型的数组的引用。
  • 同理，arry是指向一个包含10个元素的数组的引用，该数组的元素是指向int类型的指针。
重点内容2：不能使用一个数组去初始化另一个数组，也不能用一个数组去给另一个数组赋值。
重点内容3：auto 推断数组名，得到的类型是指针；decltype推断数组名，得到的对象是数组
重点内容4：所有的容器，string，数组都可使用range for。
重点内容4：多维数组

  • To use a multidimensional array in a range for, the loop control variable for all but the innermost array must be references.

    对多维数组进行遍历时，除了最里层，其余各层的循环控制变量必须为引用。

Exercise 3.27

Assuming txt_size is a function that takes no arguments and returns an int value, which of the following definitions are illegal? Explain why.

unsigned buf_size = 1024;
(a) int ia[buf_size];
(b) int ia[4 * 7 - 14];
(c) int ia[txt_size()];
(d) char st[11] = “fundamental”;

  • (a) 错误。申明数组时，数组的维度必须是常量。
  • (b) 正确。因为数组的维度是常量表达式。
  • (c) 错误。因为txt_size()返回的只是普通int，不是const int 。
  • (d) 错误。st数组的维度是11，但是用于初始化的字符串长度是12，因为还有一个空字符。

  • ia2,由于int是内置类型，且定义在函数内部，并且没有初始化，所以该变量没有值，很危险。

Exercise 3.37

const char ca[] = {‘h’, ‘e’, ‘l’, ‘l’, ‘o’};
const char cp = ca;
while (cp) {
​ cout << *cp << endl;
​ ++cp;
}

程序第一行，使用列表初始化了字符数组ca；第二行定义了一个常量指针cp,并用ca初始化；第三步的循环有问题，因为ca中没有空字符，所以循环不会终止，除非找到下一个空字符。解决方法为，用string来初始化ca，即

const char ca[] = "hello";,这样，用string初始化字符数组时会默认添加空字符。
