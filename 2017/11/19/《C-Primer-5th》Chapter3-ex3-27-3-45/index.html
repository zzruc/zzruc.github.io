Posted on 2017-11-19
重点内容1：指针数组（数组的元素是指针）和数组指针（指向数组的指针）

技巧，从右向左读：

  • 先定义了一个包含10个元素的数组ptrs，它的元素类型是int *。
  • 先定义了一个包含10个元素的数组refs,它的元素类型是int &,但是数组的元素只能是对象，不能是引用，所以此条语句错误。

技巧，从内向外读：

  • 首先读小括号里面的，可知，Parray是一个指针；再读小括号右边，可知Parray指向的是长度为10的数组；再读小括号左边，可知，该数组元素类型是int。所以，Parray是指向包含10个int类型的数组的指针。
  • 同理，可知arrRef是指向包含10个int类型的数组的引用。
  • 同理，arry是指向一个包含10个元素的数组的引用，该数组的元素是指向int类型的指针。

Exercise 3.27

Assuming txt_size is a function that takes no arguments and returns an int value, which of the following definitions are illegal? Explain why.

unsigned buf_size = 1024;
(a) int ia[buf_size];
(b) int ia[4 * 7 - 14];
(c) int ia[txt_size()];
(d) char st[11] = “fundamental”;

  • (a) 错误。申明数组时，数组的维度必须是常量。
  • (b) 正确。因为数组的维度是常量表达式。
  • (c) 错误。因为txt_size()返回的只是普通int，不是const int 。
  • (d) 错误。st数组的维度是11，但是用于初始化的字符串长度是12，因为还有一个空字符。
